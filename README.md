# MrNodeBot
## Install Steps
- npm install
- cp mv sample.env to .env
- get various api keys, refer to the .env file for information
- fill in .env file
- node index.js

### Due to this bot requiring NickServ for its validation of access levels
### It has been tested with both FreeNode and Dalnet
